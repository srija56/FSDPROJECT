Mutability

/ˌmjuː.təˈbɪl.ə.ti/

Definition

Mutability (n.): The ability of data to be modified after creation.

Example: "JavaScript objects are mutable by default."

Usage in FSD

  • React state should be treated as immutable.
  • Redux reducers must handle state immutably.
