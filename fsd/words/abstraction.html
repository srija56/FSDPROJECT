Abstraction

/æbˈstræk.ʃən/

Definition

Abstraction (n.): Hiding complex implementation details and showing only essential features.

Example: "React components use abstraction to simplify UI development."

Usage in FSD

  • APIs provide abstraction over database operations.
  • JavaScript's fetch() abstracts HTTP requests.
