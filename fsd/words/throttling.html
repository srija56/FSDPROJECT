Throttling

/ˈθrɒt.lɪŋ/

Definition

Throttling (n.): Limiting how often a function can execute.

Example: "Scroll handlers use throttling for performance."

Usage in FSD

  • Rate-limiting APIs implement throttling.
  • Lodash's _.throttle() enforces delays.
