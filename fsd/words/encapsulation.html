Encapsulation

/ɪnˌkæp.sjəˈleɪ.ʃən/

Definition

Encapsulation (n.): Bundling data with the methods that operate on that data.

Example: "Classes use encapsulation to hide internal state."

Usage in FSD

  • React hooks encapsulate component logic.
  • Private class fields (#) enforce encapsulation.
