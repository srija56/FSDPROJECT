Debounce

/diːˈbaʊns/

Definition

Debounce (v.): Delaying function execution until after a pause in events.

Example: "The search bar used debounce to reduce API calls."

Usage in FSD

  • Lodash's _.debounce() implements this pattern.
  • Used in scroll/resize event handlers.
