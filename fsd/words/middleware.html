Middleware

/ˈmɪd.əl.weər/

Definition

Middleware (n.): Software that bridges different systems or layers.

Example: "Express middleware processes requests before routing."

Usage in FSD

  • Authentication checks are implemented as middleware.
  • Redux uses middleware like Thunk for async actions.
