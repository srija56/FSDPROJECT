Dependency

/dɪˈpɛn.dən.si/

Definition

Dependency (n.): An external library or module that a project relies on.

Example: "React is listed as a dependency in package.json."

Usage in FSD

  • npm/yarn manage project dependencies.
  • Docker containers isolate dependencies.
