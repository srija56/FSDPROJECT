Cache

/kæʃ/

Definition

Cache (n./v.): Temporary storage for frequently accessed data.

Example: "The CDN cache improved website load times."

Usage in FSD

  • Browser cache stores static assets like CSS/JS.
  • Redis is used for server-side caching.
