Compilation

/ˌkɒm.pɪˈleɪ.ʃən/

Definition

Compilation (n.): Converting source code into machine-readable instructions.

Example: "TypeScript compilation catches errors before runtime."

Usage in FSD

  • Webpack bundles files during compilation.
  • Babel transpiles modern JS during compilation.
