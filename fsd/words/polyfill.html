Polyfill

/ˈpɒl.i.fɪl/

Definition

Polyfill (n.): Code that adds modern features to older browsers.

Example: "core-js provides polyfills for ES6+ features."

Usage in FSD

  • Babel includes polyfills for async/await.
  • Feature detection loads polyfills conditionally.
