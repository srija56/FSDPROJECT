Minification

/ˌmɪn.ɪ.fɪˈkeɪ.ʃən/

Definition

Minification (n.): Removing unnecessary characters from code to reduce size.

Example: "Webpack performs minification in production builds."

Usage in FSD

  • CSS/JS files are minified for faster downloads.
  • Source maps debug minified code.
