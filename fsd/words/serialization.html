Serialization

/ˌsɪə.ri.ə.laɪˈzeɪ.ʃən/

Definition

Serialization (n.): Converting objects into a storable/transmittable format.

Example: "JSON serialization prepares data for APIs."

Usage in FSD

  • REST APIs use JSON serialization.
  • Protocol Buffers optimize serialization.
