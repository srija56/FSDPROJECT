Idempotent

/ˌaɪ.dəmˈpoʊ.tənt/

Definition

Idempotent (adj.): An operation that produces the same result if executed once or multiple times.

Example: "HTTP PUT requests should be idempotent."

Usage in FSD

  • Database upsert operations are idempotent.
  • Payment processing APIs must be idempotent.
